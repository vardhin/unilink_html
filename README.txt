UNILINK HTML PROTOTYPE
link: https://vardhin.github.io/unilink_html/
QR code: unilink-qr.png (opens the live link above)


How to run:
1. Extract the ZIP file.
2. Open the folder.
3. Double-click index.html.
4. It will open in Chrome, Edge, Firefox, or another browser.

Interactive features:
- Bottom navigation
- Add money flow
- Balance update
- Payment method selection
- Reward redemption
- Transaction history
- Dark mode
- Notification toggle
- Toasts and confirmation popups

- index.html
- style.css
- script.js
- assets/mascot.png
- assets/avatar.png
- unilink-qr.png
